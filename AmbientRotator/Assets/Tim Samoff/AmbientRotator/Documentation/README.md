# Ambient Rotator System

## Quick Start
1. Add `AmbientRotator` component to any GameObject
2. Choose a Motion Profile
3. Adjust Intensity and Speed
4. Hit Play!

## Features
- 5 Motion Profiles (Subtle, Gentle, Organic, Dynamic, Chaotic)
- Custom Profile Creation with Animation Curves
- Wind System with Turbulence and Wind Zones
- Group Control with Wave Motion
- Reactive Triggers for Player Interaction
- Beat Synchronization for Music Integration
- UI Support (Position, Scale, Color)
- Physics Integration with Torque
- Preset System for Reusable Configurations
- Custom Editor Tools and Visual Debugging
- Profile Library and Browser
- Profile Blending for Complex Motion

## Creating Custom Profiles
1. Right-click in Project window
2. Select Create → Ambient Rotator → Custom Profile
3. Name your profile
4. Choose a template or start from scratch
5. Edit curves in the custom inspector
6. Preview motion in real-time

## Using Presets
1. Open Window → Ambient Rotator → Preset Browser
2. Browse available presets
3. Select a preset
4. Click "Apply to Selected"

## Performance Tips
- Use `clampMovement` to limit motion
- For many objects, use `AmbientGroup` for centralized control
- Enable `useUnscaledTime` for consistent performance
